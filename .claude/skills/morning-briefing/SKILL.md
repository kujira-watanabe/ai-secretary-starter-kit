# 朝のブリーフィング

## 目的

今日の予定とタスクを整理し、迷わず動き出せる計画を作る。

## 入力

- 現在日時
- 今日の予定（カレンダー連携・貼り付け・`inbox/` のいずれか）

## 参照

- `tasks/active.md`、`tasks/waiting.md`、`tasks/current-focus.md`
- `.claude/rules/priority-rules.md`
- `context/constraints.md`、`context/preferences.md`（朝の接し方）

## 手順

1. **現在日時を取得する**（`date` コマンド等）。時間帯に応じた挨拶とともに、日付・時刻を最初に表示する
   - 5:00〜10:59 →「おはようございます」
   - 11:00〜17:59 →「こんにちは」
   - 18:00〜翌4:59 →「こんばんは」
2. 今日の固定予定を確認する
3. 締切が近いタスク・重要タスク・返答待ちを確認する
4. 空き時間を算出する
5. 今日の最重要事項を1つ決める
6. 優先タスクを3件以内に絞る
7. 最初の30分でやることを明確にする
8. 「今日やらないこと」を明示する
9. 利用者の確認を求め、修正があれば反映して `tasks/current-focus.md` を更新する
10. **今日の計画と `tasks/active.md` のスナップショットを `logs/daily/YYYY-MM-DD.md` に記録する**（既にあれば追記、無ければ新規作成）。これは上書きされない復旧用の控え

## 出力形式

```
（挨拶）＋ YYYY年MM月DD日（曜）HH:MM 現在

■ 今日の予定
■ 今日の最重要事項（1つ）
■ 優先タスク（3件以内・実行時間帯つき）
■ 最初の30分でやること
■ 今日やらないこと
■ 返答待ちで動きがあったら教えてほしいこと
```

## 完了条件

利用者が「よし、これで始める」と言える状態。

## エラー時

予定情報が得られない場合は、タスクのみで計画を作り、「予定は未反映」と明示する。
