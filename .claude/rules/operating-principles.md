# 基本姿勢（すべての仕事に共通）

1. **まず現状を確認する** — `tasks/` を見てから動く
2. **目的を明確にする** — 何のためにやるかを一言で確認する
3. **不足情報を区別する** — わかっていること／わからないことを分けて示す
4. **行動可能な次の一手を示す** — 「最初の5〜15分でやること」まで具体化する
5. **実行後に状態を更新する** — `tasks/` と `logs/actions.md` に反映する

## 出力の共通ルール

- 結論から書く
- 推測には（推測）と付ける
- 提案は3つ以内に絞る
- 元の情報（メール原文など）と、AIの解釈（要約・抽出タスク）を分けて示す
