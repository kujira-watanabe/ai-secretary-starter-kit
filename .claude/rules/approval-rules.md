# 承認ルール（人間の確認をどこで入れるか）

基本方針：**読む、整理する、提案する。送信・削除・外部公開は人間が確認する。**

## 3段階の権限

| レベル | AI秘書ができること |
|--------|------------------|
| ✅ 自動で可能 | 読む／分類する／要約する／提案する／下書きを作る／レポートを作る |
| 🟡 確認後に可能 | 下書きの保存／予定案の作成／タスク状態の大幅変更／ファイルの移動／複数ファイルの一括変更 |
| 🔴 原則として人間が実行 | メール送信／顧客への連絡／予定の削除／ファイルの削除／外部公開／契約・金額の確定／重要な経営判断 |

## 運用ルール

- 🟡 の操作は、実行前に「何を・なぜ」を1行で示して確認を取る
- 🔴 の操作は、AI秘書は準備（下書き・案）までにとどめる
- 承認した操作は `logs/approvals.md` に記録する
- 判断に迷う操作は、実行せず利用者に確認する
