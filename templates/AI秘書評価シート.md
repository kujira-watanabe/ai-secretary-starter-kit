|---|------|----|------|

| 3 | 優先順位は自分の感覚と合っているか | | |

| 5 | 自分で判断してよい範囲を守っているか（🔴操作に踏み込んでいないか） | | |
| 6 | 出力はそのまま使えるか（形式・長さ・トーン） | | |
