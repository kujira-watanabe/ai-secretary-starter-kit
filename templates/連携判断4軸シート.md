# 外部連携の判断4軸シート（Day2）

**つなげられることと、つないでよいことは別。** Gmail・カレンダー等を連携する前に、4つの軸で確認する。

## 1. 誰が管理するアカウントか

- [ ] 個人管理 → 自分の判断で連携してよい
- [ ] 個人事業用 → 自分の判断で連携してよい
- [ ] 法人管理（自分が経営者） → 経営者として判断
- [ ] 会社支給 → **会社の管理者に確認が必要**
- [ ] 顧客管理 → **顧客との契約確認が必要**

## 2. どんな情報が含まれるか

- [ ] 公開情報のみ
- [ ] 自分の個人情報
- [ ] 顧客情報・取引情報 → 慎重に
- [ ] 契約・未公開情報 → 原則つながない
- [ ] 認証情報（パスワード等） → **絶対に扱わせない**

## 3. 誰が利用可否を判断できるか

本人で判断できる → 連携可。会社規程・顧客契約の確認が必要 → 確認が取れるまで**手動入力（貼り付け）で運用**。

## 4. どこまで権限を与えるか

```
読む → 検索する → 要約する → 提案する → 下書きを作る
（ここまでが基本）─────────────────────────
登録する → 変更する → 削除する → 送信する
（ここからは人間が確認）
```

## 基本方針

> **読む、整理する、提案する。送信・削除・外部公開は人間が確認する。**

連携しなくても、メール本文の貼り付け・予定の手入力で同じ仕事はできる。迷ったらつながない。
