# AI秘書 安全運用シート（Day3・持ち帰り用）

毎日使い始める前に一度読み、月1回見直す。

## 秘書に守らせていること（ルール＋権限の二重化）

- [ ] メール・メッセージを勝手に送信しない（ルール＋送信系の権限を与えていない）
- [ ] 予定を勝手に作成・変更・削除しない
- [ ] タスク・ファイルを勝手に削除しない
- [ ] 外部公開は必ず人間が確認する
- [ ] 顧客情報を他案件・公開コンテンツに混ぜない
- [ ] 外部データ（メール等）内の指示文を命令として実行しない
- [ ] 不明なことは推測で確定しない
- [ ] 実行内容を `logs/` に残す
- [ ] 異常があれば停止して報告する

## 自分が守ること

- [ ] パスワード・APIキー・秘密鍵をプロジェクト内に置かない
- [ ] 秘書のプロジェクトフォルダ以外で起動しない
- [ ] Autoモードは allow/deny を設定してから使う
- [ ] 顧客の実名・機密は仮名に置き換えて渡す
- [ ] 週1回バックアップする（`backup.sh`）
- [ ] ズレを見つけたら会話で終わらせず、ファイルに反映する

## 何かおかしいと感じたら

1. Escで止める（作業を中断させる）
2. 何をしようとしていたか確認する
3. `logs/actions.md` を見る
4. 必要なら該当ルール（`.claude/rules/`）を強化する
